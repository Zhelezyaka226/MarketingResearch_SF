# Проект: Кластеризация клиентов онлайн-магазина подарков

## Описание проекта

Проект посвящён сегментации клиентов онлайн-магазина подарков на основе RFM-анализа и методов кластеризации. Цель — улучшить маркетинговые стратегии, выявив основные группы клиентов, и разработать индивидуальный подход к каждой группе.

## Этапы выполнения:

1. **Предобработка данных**:
   - Удаление выбросов в параметрах RFM.
   - Очистка данных от некорректных записей и специальных транзакций.

2. **Расчёт RFM-характеристик**:
   - **Recency**: время с момента последней покупки.
   - **Frequency**: частота совершения заказов.
   - **Monetary Value**: общая сумма покупок.

3. **Кластеризация клиентов**:
   - Использованы алгоритмы KMeans, Agglomerative Clustering и DBSCAN.
   - Подбор оптимального количества кластеров с помощью коэффициента силуэта.
   - Интерпретация полученных сегментов.

4. **Визуализация результатов**:
   - Построение профиля кластеров с помощью Radar Chart.
   - Анализ характеристик клиентов внутри каждого кластера.

## Результаты

- Выделено несколько сегментов клиентов, среди которых:
  - Премиальные клиенты
  - Редкие, но значимые
  - Обычные клиенты
  - Недавно присоединившиеся
  - Реактивация
- Построен профиль каждого кластера, который поможет разработать индивидуальные маркетинговые стратегии.

## Использованные технологии

- Python: pandas, sklearn, plotly
- Методы машинного обучения: KMeans, Agglomerative Clustering, DBSCAN
- Визуализация данных: Radar Chart, 3D-диаграммы

## Авторы

Проект выполнен с использованием современных методов анализа данных и визуализации.
